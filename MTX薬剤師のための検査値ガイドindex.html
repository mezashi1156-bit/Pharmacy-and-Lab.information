MECHANISM 作用機序と薬剤師が押さえるべき全体像
①葉酸拮抗作用（DHFR阻害）
ジヒドロ葉酸還元酵素（DHFR）を阻害 → テトラヒドロ葉酸（活性型葉酸）産生↓ → DNA合成・細胞分裂抑制。
増殖の速い細胞（滑膜細胞・炎症細胞・造血細胞・消化管粘膜）に強く影響 → 血液毒性・消化管障害・口内炎の主な背景機序。葉酸製剤（ロイコボリン等）の併用で副作用が軽減される。
②抗炎症作用（アデノシン放出促進）
AICAR transformylase阻害 → 細胞外アデノシン↑ → 好中球遊走・炎症性サイトカイン（IL-1）・コラゲナーゼ産生・血管新生・滑膜増生を抑制（リウマトレックスIF）。
→ CRP・ESR・MMP-3の低下として検査値に反映される。
③腎排泄とアルブミン結合——副作用リスクの根幹
MTXは主に腎臓から未変化体で排泄（尿中69.5%）（リウマトレックスIF）。腎機能低下でMTX蓄積 → 全副作用が急増。
また血清アルブミンと高度に結合しており、低アルブミン血症では遊離型MTXが増加し毒性が増強する。

→ 薬剤師の役割は「効果の確認（炎症指標の改善）」と「毒性の早期発見（血液・肝・腎・肺）」の両面からの検査値モニタリングにある。

EFFICACY 関節リウマチ治療効果を評価する検査
疾患活動性スコア（DAS28・SDAI・CDAI） 低下＝改善 主要評価指標
各スコアの構成
DAS28-CRP = 0.56×√圧痛関節数 + 0.28×√腫脹関節数 + 0.36×ln(CRP+1) + 0.014×患者VAS + 0.96
SDAI = 圧痛関節数 + 腫脹関節数 + 患者VAS + 医師VAS + CRP(mg/dL)
CDAI = 圧痛関節数 + 腫脹関節数 + 患者VAS + 医師VAS（血液検査不要）

モニタリング頻度（MTX手引き2023）：治療開始後は4〜8週ごと。寛解または低疾患活動性を3カ月以上維持後は評価間隔の延長可。

国内臨床試験成績（リウマトレックスIF）：国内46施設・204例のRA患者を対象とした臨床試験で、6mg/週投与群の改善率（著明改善＋中等度改善）は60.4%（32/53例）。

DAS28 < 2.6（または SDAI ≤ 3.3）→ 寛解 DAS28 2.6〜3.2 → 低疾患活動性 DAS28 > 5.1 → 高疾患活動性・治療見直し
CRP・ESR（赤血球沈降速度） 低下＝炎症鎮静化

CRPは急性相タンパク（数時間〜1日で変動）、ESRはフィブリノゲン増加を反映（数日〜週単位で変動）。DAS28算出にはCRP版とESR版の2種類があるため、どちらが使われているか把握しておく。

モニタリング頻度（MTX手引き2023）：開始・増量後3カ月以内は2〜4週ごと、以降は4〜12週ごと。

CLINICAL PHARMACIST POINT
MTX投与中に炎症改善なくCRPが上昇している場合、疾患再燃だけでなく感染症（日和見感染含む）の可能性も高い。感染症はMTX継続の可否に直結するため、単純に「疾患悪化」と判断せず医師への情報提供が重要。
MMP-3（マトリックスメタロプロテアーゼ-3） 低下＝滑膜炎の鎮静化 12週ごと算定

滑膜細胞が産生する軟骨分解酵素で、関節破壊リスクの補助的指標。CRPが陰性化していてもMMP-3が高値の場合は滑膜炎が持続している可能性がある（安田・小原, 臨床リウマチ Vol.24 No.2）。

保険上の算定は12週（約3カ月）ごとまで。

低下継続 → 治療奏効・関節破壊リスク低下 CRP陰性化後も高値継続 → 画像（X線・関節エコー）での関節評価を
RF（リウマトイド因子）/ 関節単純X線・HAQ-DI 24〜52週ごと / 年1回

RF：CRP・ESRと異なり治療効果の反映が緩徐（数カ月〜1年以上）。短期間での評価は意義が低い。

関節X線（手・足部）：骨びらん・関節裂隙狭小化の進行抑制を年1回評価。有効な治療では骨破壊の進行が抑制される（構造的寛解の評価）。

HAQ-DI：患者自己申告による身体機能評価。検査値の改善と身体機能改善の間にタイムラグがあることを念頭に置く。24〜52週ごと。

SAFETY 副作用を評価する検査・発現傾向・対応の目安

★ 発現傾向はリウマトレックスIF（第4版）・MTX診療手引き2023年版より。重大な副作用は死亡例が報告されているため特に注意。

末梢血液検査（CBC）/ WBC・好中球・PLT・RBC・MCV 最重大 死亡報告34.6%
骨髄抑制の機序と傾向
DHFR阻害 → 造血幹細胞のDNA合成障害 → 汎血球減少・無顆粒球症。国内死亡報告（2012年まで465例）のうち血液障害161例（34.6%）が最多。汎血球減少97例・骨髄抑制39例・白血球減少18例を含む（MTX適正使用情報Vol.18 / 大阪大学）。

MCVに注目：葉酸拮抗により赤血球の大球性変化（MCV上昇）が骨髄抑制に先行して出現することがある。早期の薬物毒性サインとして把握しておく。

前駆症状（添付文書）：発熱・咽頭痛・インフルエンザ様症状が先行することがある。患者への服薬指導で必ず伝える。

WBC < 4,000/mm³ → 慎重経過観察・医師へ報告 WBC < 3,000/mm³ または PLT < 50,000/mm³ → 投与禁忌水準・即時対応

腎機能低下・低アルブミン血症・NSAIDs複数内服でリスクが急増する。

肝機能（AST・ALT・ALP・γGTP・アルブミン） 重大 臨床検査値異常23.8%
肝障害の機序と傾向
MTXが肝細胞内でポリグルタミン酸化されて蓄積 → 直接的な肝細胞毒性 → AST・ALT上昇。長期投与で肝線維化・肝硬変のリスクがある。飲酒との相乗で顕著に悪化。国内臨床試験（204例）で臨床検査値異常発現率23.8%（15/63例）、最多が肝機能関連（リウマトレックスIF）。

アルブミン低下の意味：アルブミン<3.0 g/dLでは遊離型MTXが増加し毒性リスクが直接増大する。MTX手引き2023では「慎重投与水準」として明示。

HBV・HCV感染者の注意：MTX投与中および投与終了後もHBV-DNA・肝炎ウイルスマーカーのモニタリングが必要。HBV再活性化による重篤な肝炎・死亡例が報告されている（添付文書）。

AST/ALT が基準値上限の2倍超 → 減量・中止検討の目安（MTX手引き2023） 急激な肝機能悪化 + アルブミン低下 → 毒性増強リスクとして即時対応

飲酒歴・NAFLD（脂肪肝）の確認も重要。ALT上昇時はAST/ALT比も参考に。

腎機能（BUN・Cr・eGFR）/ 尿一般検査 重大 全副作用増強の引き金
腎機能低下がなぜ危険か
MTXは尿中排泄69.5%（リウマトレックスIF）のため、eGFR低下 → MTX蓄積 → 骨髄抑制・肝障害など全副作用が連鎖的に増悪する。「腎機能の悪化が全ての副作用リスクを増幅する」という連鎖を患者・他職種に説明できることが重要。

投与禁忌・慎重投与水準（MTX手引き2023）：

　GFR < 30 mL/分/1.73m² → 投与禁忌の目安
GFR < 60 mL/分/1.73m² → 慎重投与（低用量から開始）

脱水・造影剤・NSAIDs使用などで腎機能が一時的に悪化した際もMTX蓄積リスクとして意識する。

eGFR 30〜60 → 慎重投与・頻回モニタリング・必要時減量 eGFR < 30 または急激なCr上昇 → 禁忌水準・即時報告・休薬検討
胸部X線・KL-6・SP-D（肺障害モニタリング） 重大 死亡報告28.2%
MTX肺炎の特徴
発症機序は主に過敏反応（詳細不明）。用量非依存性（少量でも発症しうる）。国内調査では75%が服用開始半年以内に発症するが、数年〜十数年後の発症例もある（日本呼吸器学会 薬剤性肺障害手引き2013）。国内死亡報告のうち肺障害131例（28.2%）、うち間質性肺炎120例（大阪大学）。

リスク因子：60歳以上・胸膜病変・低アルブミン血症・糖尿病・既存の肺病変（Ann Intern Med 1997;127:356-64）。

典型症状（患者への服薬指導で必ず伝える）：38℃以上の発熱・乾性咳嗽・安静時の息苦しさ。「風邪と思って3日改善しなければ受診」では遅い可能性があるため、発症したら即受診を指導する。

KL-6・SP-Dの位置付け：MTX手引き2023では「参考程度にとどめる」と記載。スクリーニングの主体は症状と胸部画像にある。

KL-6上昇 → 参考情報として報告、症状と画像評価を促す 発熱 + 呼吸器症状 → MTX即時中止・医師へ緊急報告
CLINICAL PHARMACIST POINT
発熱＋呼吸器症状時は①感染性肺炎②ニューモシスティス肺炎③MTX肺炎の三者鑑別が必要。いずれも重篤になりうるため、自己判断で服薬継続させず即受診を優先する。
感染症マーカー（β-Dグルカン・HBV関連・結核関連） 重大 死亡報告17.8%

MTXの免疫抑制作用により日和見感染リスクが上昇。国内死亡報告のうち感染症83例（17.8%）。ニューモシスティス肺炎25例・肺炎19例・敗血症11例など（大阪大学）。

HBV再活性化：HBsキャリアや既往感染者（HBc抗体陽性）では、MTX投与中・終了後もHBV-DNAのモニタリングが必要。再活性化が起きると重篤な肝炎を引き起こし死亡例あり（添付文書）。

結核（潜在性）：投与前のQuantiFERON/Tスポット検査が必須。陽性時はINH（イソニアジド300mg/日）予防投与を先行してからMTX開始（MTX手引き2023）。

HBV-DNA上昇 → MTX中止・抗ウイルス薬開始・消化器科へ
可溶性IL-2受容体（sIL-2R）・LDH・CBC 重大 平均7年後に発症

MTX関連リンパ増殖性疾患（LPD）。EBV関連が多く、MTX中止で自然退縮する例があるのが特徴。60〜70歳代の女性に多く、MTX開始から発症まで平均7年（ゼルヤンツLPD資料）。

手がかりとなる検査パターン：関節炎の活動性が落ち着いているにもかかわらずCRP上昇・sIL-2R高値という乖離が生じた場合にLPDを疑う。リンパ節腫大・B症状（発熱・寝汗・体重減少）の有無も確認。

sIL-2R上昇 + 関節炎落ち着き + CRP上昇 → LPDの可能性を医師へ LDH上昇 + リンパ球増多 + sIL-2R著増 → 悪性リンパ腫として精査
消化管障害（口内炎・悪心・下痢）/ 栄養状態 頻度高い 服薬継続に影響

葉酸拮抗による消化管粘膜への直接毒性。MTXの副作用のなかで最も頻度が高く、服薬アドヒアランス低下の主因となる。

対処法：葉酸製剤（フォリアミン等5mg）のMTX最終服用24〜48時間後投与で軽減可能。悪心には制吐剤（メトクロプラミド等）の併用も考慮。

栄養状態の確認指標：アルブミン・プレアルブミン・体重変化。消化管症状が強い場合はMTX吸収にも影響するため経過を注視する。

口内炎悪化・食欲不振の継続 → 葉酸補充・用量調整を医師に提案
