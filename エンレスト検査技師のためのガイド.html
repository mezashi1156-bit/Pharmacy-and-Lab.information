臨床検査技師のための機序・解釈・報告ポイント

PARADIGM-HF・PARALLEL-HF・PIONEER-HF・PANORAMA-HF 各試験データ + エンレストIF（第8版, 2024年3月）に基づく
NEP阻害（サクビトリル） ARB（バルサルタン） 50 / 100 / 200 mg 錠 1日2回投与
MECHANISM まず理解しておく作用機序と検査値への影響の全体像
サクビトリル（NEP阻害）の作用
ネプリライシン（NEP）を阻害 → ANP・BNP・ブラジキニンの分解が抑制 → 利尿・血管拡張・心保護。
⚠ BNPは分解されずに血中に蓄積するため、心不全改善後も血漿BNP値は高値を示しうる（偽高値）。
バルサルタン（ARB）の作用
AT1受容体遮断 → RAAS抑制 → アルドステロン分泌↓、ナトリウム排泄↑、血圧↓、腎保護（糸球体内圧低下）。
ただし輸出細動脈拡張により開始初期にeGFRが一時的に低下（hemodynamic dip）する。

→ この2つの機序が組み合わさることで、検査値には「治療効果として動く変化」と「副作用・介入が必要な変化」が混在する。判別が臨床検査技師の報告の要点となる。

EFFICACY 治療効果を評価する検査とその解釈
NT-proBNP 低下＝改善 主要指標
なぜNT-proBNPを使うか
NT-proBNPはネプリライシンの基質ではない。サクビトリルによるNEP阻害の影響を受けず、心負荷を忠実に反映する。BNPとは対照的に「薬の影響を受けない心不全マーカー」として機能する。

試験結果：PIONEER-HF試験では、ARNi群のNT-proBNP低下率が46.7%に対しACEi群は25.3%（p<0.0001）。群間差は開始1週間後の時点ですでに有意差あり。

PARADIGM-HF・PARALLEL-HF・PANORAMA-HF（小児）いずれにおいても、エンレスト群でNT-proBNPの低下が確認されている（エンレストIF 第8版）。

低下継続 → 治療奏効 横ばい → 増量・他薬追加を検討 上昇 → 心不全増悪を疑う
BNP（血漿） 偽高値に注意
⚠ 解釈の落とし穴
BNPはネプリライシンの基質。NEP阻害によりBNPの分解が抑制されるため、心不全が改善していても血漿BNP値は高値を維持・上昇する。エンレスト投与中のBNP単独評価は不適切。

報告の際は「エンレスト投与中のBNP値は NEP阻害の影響で偽高値を示しうる。NT-proBNPでの評価を推奨」とコメントを付記することが有用。

REPORTING POINT
BNP高値≠心不全増悪。NT-proBNPとのセットで報告し、解釈の注意を促す。
LVEF（心エコー） 改善期待

PARADIGM-HF試験の組み入れ基準はLVEF≤40%（HFrEF）。治療効果として逆リモデリング（心室容積の縮小・LVEF上昇）が期待される。心エコーは心筋リモデリング抑制の構造的評価として用いられる。

（出典：PARADIGM-HF試験 N Engl J Med 2014、エンレストIF 第8版）

血漿レニン活性・血清アルドステロン 上昇（薬理効果）
上昇の機序
ARBによるAT1遮断 → フィードバックでレニン分泌が代償性に増加。アルドステロンも用量依存的に増加（ARBによるAT2刺激・RAAS活性化）。

インタビューフォームでは、エンレスト50mg群・100mg群でプラセボ群と比較してアルドステロンが有意に増加（p≤0.0016、共分散分析）。レニン濃度は全用量群で有意に上昇（p≤0.0071）。

→ 「アルドステロン・レニン高値」はエンレストの薬理効果として予期される変化。副腎疾患との鑑別が必要な場合は服薬情報の確認を。

NT-proANP（心房性 NUPペプチド） 上昇（NEP阻害効果）

ANPもネプリライシンの基質であり、NEP阻害により血中ANP・NT-proANP は上昇する。インタビューフォームにも評価項目として記載。BNPと同様に「上昇が必ずしも病態悪化を示さない」点に注意（エンレストIF 第8版）。

SAFETY 副作用を評価する検査・発現率・判断基準

★ 発現率は添付文書（日経メディカル処方薬事典、エンレストIF 第8版）より。重大な副作用には特に注意。

血圧（SBP / DBP） 重大 8.8%
低下の機序
NEP阻害によるNUPペプチド増加（血管拡張・利尿）＋ ARBによる血管拡張 の相乗効果。バルサルタン単独より収縮期血圧を約5〜7 mmHg追加で低下させる（PARADIGM-HF副次解析）。

PARADIGM-HF試験： 導入期間中の低血圧による試験中止はARNI群1.7%（ほぼACEi群と同等）。しかし開始後の減量原因の第1位は血圧低下（ARNI 21%、ACEi 16%、p<0.001）。

国内PARALLEL-HF試験では50mg×2回から導入しSBP変動は-4.3±12.3 mmHgと個人差が大きい。

SBP < 100 mmHg → 減量検討 ショック・失神（0.1%未満〜0.2%）→ 即時対応

高リスク：高齢者、ベースラインSBP低値、eGFR低値、NT-proBNP高値患者（PARADIGM-HF解析）

血清カリウム（K） 重大 3.9%
上昇の機序
ARBによるアルドステロン抑制 → 腎尿細管でのK排泄低下 → 高カリウム血症。MRA（スピロノラクトン・エプレレノン）との併用でリスクが最も増大する。

添付文書では高カリウム血症のリスク因子として、腎機能障害・糖尿病・低アルドステロン症・高カリウム食を明示（今日の臨床サポート エンレスト錠50mg）。

PIONEER-HF試験では、ARNi群とACEi群で高カリウム血症の発現率に有意差なし。

K ≥ 5.5 mEq/L → 減量・中止検討 K ≥ 6.0 mEq/L → 即時対応（心電図確認含む）

⚠ SGLT2阻害薬との4剤併用（いわゆる「fantastic four」）時はSGLT2のK低下効果が一部相殺するが、依然として高K注意。

Cr（クレアチニン）・eGFR 重大 腎機能障害2.4% / 腎不全0.6%
変動の2フェーズ
【開始初期】 輸出細動脈拡張 → 糸球体内圧低下 →「hemodynamic dip」としてeGFRが10〜20%低下、Crが軽度上昇。これは薬理的に予期される変化。
【中長期】 糸球体過剰濾過の改善 → 腎保護効果。ただし真の腎機能悪化との鑑別が必要。

添付文書では、eGFR 30未満の重度腎障害患者に対しては低用量からの開始と厳重なモニタリングを規定（今日の臨床サポート エンレスト錠50mg）。

開始後Cr軽度上昇 → hemodynamic dipを疑い、経過観察 急激なCr上昇・乏尿 → 腎不全として対応

BUN/Cr比の上昇も脱水の指標として合わせて確認。

Ht / Hb・BUN / Cr比・尿酸（UA） 国内で特に注意
国内データに注意
PARALLEL-HF試験（国内）では脱水関連事象がエンレスト群16.22% vs エナラプリル群7.14%と、国内で明らかな差。海外PARADIGM-HFでは両群ほぼ同等だったため、日本人での注意喚起が特に設定されている（エンレストIF 第8版）。

NEP阻害によるナトリウム利尿促進が、利尿薬との相乗で体液量を過剰に低下させうる。

検査値の変動傾向：

　Ht・Hb → 上昇（血液濃縮）/ UA → 上昇（尿酸排泄低下）/ BUN/Cr比 → 上昇（腎前性的変化）

Ht・UA・BUN/Cr比の同時上昇 → 脱水を疑う
血管性浮腫（臨床的評価） 重大 0.2%
機序：36時間ルールの根拠
NEP阻害によりブラジキニンの分解が抑制される。ACE阻害薬はブラジキニン分解を阻害する別機序を持つため、両剤が重複すると重篤な血管性浮腫リスクが激増。ACEiからの切替には36時間以上の休薬期間が必須。

検査値での早期検出は困難だが、C1インヒビター欠損症（遺伝性血管性浮腫）患者では特にリスク高。補体系の評価（C1q・C4等）が参考になる場合がある。

舌・咽頭腫脹、喘鳴、腹痛などの臨床症状が主要な判断根拠。

血算（WBC・好中球・血小板） 重大 低頻度〜頻度不明

無顆粒球症（頻度不明）・白血球減少（0.1%未満）・血小板減少（頻度不明）が重大な副作用として設定されている（日経メディカル処方薬事典）。

明確な機序は不明だがRAAS関連薬クラスエフェクト。定期的な血算のフォローで早期検出を狙う。

好中球減少・血小板低下 → 薬剤性を念頭に報告
肝機能（AST・ALT）・CK（CPK） 重大（低頻度）

肝炎・横紋筋融解症（CK上昇・血中ミオグロビン上昇・尿中ミオグロビン上昇）が重大な副作用として設定されている。いずれも低頻度だが、スタチン等との多剤併用患者ではCKモニタリングが重要となる場合がある（日経メディカル処方薬事典）。

血糖・HbA1c 重大（頻度不明）/ 好影響の報告も

低血糖が重大な副作用として設定されている（頻度不明）。一方、サクビトリル/バルサルタンが糖代謝（HbA1c）に改善方向の影響を示した研究報告もあるが、対象・条件により結果は異なる（PMC掲載論文、エンレスト解説記事）。

糖尿病患者では低血糖・高カリウム血症の両リスクが高まるため、血糖・K・eGFRのトリプルモニタリングが推奨される。
